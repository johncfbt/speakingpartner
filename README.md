# SpeakingPartner MERN app

This project use MERN (MongoDB, Express, React, NodeJS) stack.<br>

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run dev
```
